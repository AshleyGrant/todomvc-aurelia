## Local Installation

Requirements:

- [NodeJS](http://nodejs.org/download/)
- [jspm](http://jspm.io/)
- [gulp](http://gulpjs.com/)

Clone the repository:

    git clone https://github.com/mhoyer/todomvc-aurelia

Install jspm and npm packages:

    npm install
    jspm install

Run tests and HTTP-server:

    gulp watch
